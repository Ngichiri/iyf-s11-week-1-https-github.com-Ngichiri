## Project Description
I built a basic webpage called "Web Foundation" using only HTML to practice document structure, headings, paragraphs, lists, links, and images. This project taught me how browsers read HTML and how to organize content properly before adding CSS.

## Technologies Used
- HTML5

## Features
- Feature 1: Navigation menu with Home, About, Projects, Contact links
- Feature 2: Profile section with name, photo, and introduction paragraph
- Feature 3: Hobbies list showing Coding & Web Development, Robotics & Electronics, Music & Audio Production

## How to Run
1. Clone this repository
2. Open index.html in your browser

## Lessons Learned
I learned how to use semantic HTML tags like 
 <nav>, <h1>, <p>, <ul>, <li> to structure content properly.
I also learned that every opening tag needs a closing tag.

## Challenges Faced
My image wasn’t displaying at first. I fixed it by putting the image file in the same folder as index.html and correcting the src path.
